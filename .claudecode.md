# infinite-specs-xr-core | Agent Instructions

## Project Context
This is an Android XR architectural research sandbox exploring "Strange Loops" in system engineering. It connects physical environment telemetry (gaze/voice) to autonomous coding agents using the Model Context Protocol (MCP).

## Tech Stack & Constraints
- **Platform**: Android XR (Developer Preview 4).
- **SDKs**: `androidx.xr.compose`, `androidx.xr.scenecore`, `androidx.xr.runtime`.
- **Language**: Kotlin 2.0.21.
- **Build**: AGP 8.9.1, Gradle 8.11.1, Java 17.
- **Transport**: Ktor SSE (Server-Sent Events) via MCP server daemons.

## Coding Standards
- **Style**: Use `ktlint` rules.
- **Formatting**: Always run `./gradlew spotlessApply` before finalizing code changes.
- **HUD UI**: HUD components should use `FontFamily.Monospace` and the `HudColors` palette defined in `SpatialPanelComposable.kt`.
- **Trailing Commas**: Mandatory for all multi-line parameters/arguments.
- **Version Control**: Always use **Conventional Commits** for all git commit messages (e.g., `feat:`, `fix:`, `build:`, `docs:`). Ensure commit messages are descriptive.


## XR Design Patterns
- **Subspaces**: All spatial content must be wrapped in a `Subspace` block.
- **Spatial Panels**: Use `SpatialPanel` with `SubspaceModifier.transformingMovable()` for main HUD elements.
- **Orbiters**: Use `Orbiter` with `OrbiterAnchorPoint` for secondary controls (buttons, menus) anchored to panels.
- **Full Space**: The app is configured to launch in `XR_ACTIVITY_START_MODE_FULL_SPACE_MANAGED`. Ensure spatial UI is compatible with this mode.

## Workflow Rules
1. **Roadmap Alignment**: Always check `ROADMAP.md` before starting a new feature to ensure alignment with the current Phase.
2. **Build First**: Before proposing deep logic changes, verify the build with `./gradlew assembleDebug`.
2. **Lint**: Never commit without running `./gradlew spotlessApply`.
3. **Tests**: Use `StrangeLoopTest.kt` to verify end-to-end telemetry-to-specification mapping.
4. **Imports**: Do not add unused imports. Spotless will strip them, but avoid introducing them in snippets.
